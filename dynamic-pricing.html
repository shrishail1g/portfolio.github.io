Flipkart Business Case

Dynamic Pricing & Discount Optimization

üìå Business Case

Flipkart wants to adjust product prices dynamically based on demand, inventory levels, competitor prices, and customer behavior.

üß† Algorithms Used

  • Multi-Armed Bandit (UCB, Thompson Sampling): Choose optimal price point based on exploration and exploitation.
  • Price Elasticity Estimation: Regression-based models to estimate how demand responds to price changes.
  • Game Theory: Model competitor behavior in pricing wars and determine optimal strategies.

üß† Intuition

Like a shopkeeper lowering prices on umbrellas before rain, Flipkart must adapt to market signals to stay competitive and maximize revenue.

üí• Impact

Increased revenue, improved inventory turnover, better competitiveness in high-demand situations, and greater buyer satisfaction during flash sales.

üíª Sample Base Code (Thompson Sampling - C++)

#include <iostream>
#include <random>
#include <vector>
using namespace std;

int main() {
    int N = 10000; // number of users
    int d = 2; // number of price variants (arms)
    vector rewards1(N);
    vector rewards2(N);
    default_random_engine generator;
    bernoulli_distribution dist1(0.2); // arm 1
    bernoulli_distribution dist2(0.5); // arm 2

    int number_of_rewards_1[d] = {0};
    int number_of_rewards_0[d] = {0};
    vector ads_selected;

    for (int n = 0; n < N; n++) {
        int ad = 0;
        double max_random = 0.0;
        for (int i = 0; i < d; i++) {
            double random_beta = beta_distribution<>(number_of_rewards_1[i] + 1, number_of_rewards_0[i] + 1)(generator);
            if (random_beta > max_random) {
                max_random = random_beta;
                ad = i;
            }
        }
        ads_selected.push_back(ad);
        int reward = (ad == 0) ? dist1(generator) : dist2(generator);
        if (reward == 1)
            number_of_rewards_1[ad]++;
        else
            number_of_rewards_0[ad]++;
    }

    cout << "Rewards for Arm 0: " << number_of_rewards_1[0] << endl;
    cout << "Rewards for Arm 1: " << number_of_rewards_1[1] << endl;
    return 0;
}

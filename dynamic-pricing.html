Dynamic Pricing & Discount Optimization

ðŸ“Œ Business Case

Adjusting product prices dynamically in real-time is crucial for platforms like Flipkart. By analyzing demand, stock levels, and competitor prices, we can optimize pricing strategies to maximize revenue and conversion while preventing stockouts.

ðŸ§  Algorithms Used

  • Multi-Armed Bandit (MAB) - Thompson Sampling, UCB
  • Price Elasticity Estimation - Linear Regression to measure how demand responds to price changes
  • Game Theory - Competitive pricing analysis with Nash Equilibrium assumptions

ðŸ§  Intuition

Imagine a shopkeeper quickly lowering the price of umbrellas when it starts to drizzle. Similarly, algorithms should respond to rising demand or excess supply. If competitor prices drop, our price should react instantly.

ðŸ§ª Sample C++ Code (Thompson Sampling Example)

#include <iostream>
#include <random>
#include <vector>
using namespace std;

int main() {
    int n_rounds = 1000, n_arms = 2;
    vector<int> successes(n_arms, 1), failures(n_arms, 1);
    default_random_engine generator;
    bernoulli_distribution arm1(0.5);  // true CTR 50%
    bernoulli_distribution arm2(0.7);  // true CTR 70%

    for (int t = 0; t < n_rounds; ++t) {
        vector<double> sampled;
        for (int i = 0; i < n_arms; ++i) {
            gamma_distribution<double> dist(successes[i], 1.0 / (failures[i]));
            sampled.push_back(dist(generator));
        }
        int chosen_arm = sampled[0] > sampled[1] ? 0 : 1;
        bool reward = (chosen_arm == 0 ? arm1(generator) : arm2(generator));

        if (reward)
            successes[chosen_arm]++;
        else
            failures[chosen_arm]++;
    }

    cout << "Arm 0 Successes: " << successes[0] << ", Failures: " << failures[0] << endl;
    cout << "Arm 1 Successes: " << successes[1] << ", Failures: " << failures[1] << endl;
    return 0;
}

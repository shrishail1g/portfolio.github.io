Shopping Cart Price Calculation

üìå Business Case

When users add multiple items to their cart, Flipkart must calculate final prices with GST, discounts, coupons, and delivery charges applied instantly.

üß† Algorithms Used

  • Greedy Algorithms: Apply best discount offers in order of impact.
  • Dynamic Pricing Rules: Modular functions handle GST, delivery zones, coupon policies.
  • Design Patterns: Strategy and Composite for flexible discount rule chaining.

üß† Intuition

Flipkart's checkout logic needs to mimic how a smart cashier would apply the best set of discounts and taxes to your cart in real-time.

üí• Impact

Improves cart conversion rates, reduces payment mismatches, and ensures better coupon utilization without overloading pricing engines.

üíª Sample Base Code (Simple Pricing Logic - C++)

#include <iostream>
#include <vector>
using namespace std;

struct Item {
    double price;
    double discount;
    double gst;
};

double calculateFinalPrice(const vector<Item>& items) {
    double total = 0.0;
    for (auto &item : items) {
        double discounted = item.price - item.discount;
        double taxed = discounted + (discounted * item.gst / 100.0);
        total += taxed;
    }
    return total;
}

int main() {
    vector<Item> cart = {
        {1000, 200, 18},
        {500, 50, 12},
        {1500, 300, 5}
    };

    cout << "Final Cart Price: Rs." << calculateFinalPrice(cart) << endl;
    return 0;
}

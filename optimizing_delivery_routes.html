Flipkart Business Case

Optimizing Delivery Routes (Graph Algorithms)

üìå Business Case

Flipkart must ensure that delivery agents cover multiple customer locations efficiently in a single trip. This requires computing the optimal path considering traffic conditions, distances, and timing constraints to reduce cost and maximize customer satisfaction.

üîß Algorithms Used

  • Dijkstra's / A*: Compute the shortest path between delivery points with weights representing travel time or cost.
  • Floyd-Warshall: Useful for precomputing all-pairs shortest paths in dense delivery graphs.
  • Traveling Salesman Problem (TSP): Optimize the route of a single delivery agent visiting multiple locations.
  • Minimum Spanning Tree (MST): Used to connect multiple delivery hubs with minimum infrastructure cost.

üß† Intuition

Imagine the delivery agent as a traveler who wants to complete all deliveries using the least distance and time. The challenge is similar to a tourist planning the most efficient sightseeing route.

üí• Impact

Reduced fuel usage, minimized delivery times, improved delivery success rates, and better utilization of agent capacity.

üíª Sample Base Code (TSP with DP - C++)

#include <iostream>
#include <vector>
#include <limits.h>
using namespace std;

const int INF = 1e9;
int tsp(int mask, int pos, vector>& dist, vector>& dp, int n) {
    if (mask == ((1 << n) - 1))
        return dist[pos][0];

    if (dp[mask][pos] != -1)
        return dp[mask][pos];

    int ans = INF;
    for (int city = 0; city < n; city++) {
        if ((mask & (1 << city)) == 0) {
            int newAns = dist[pos][city] + tsp(mask | (1 << city), city, dist, dp, n);
            ans = min(ans, newAns);
        }
    }
    return dp[mask][pos] = ans;
}

int main() {
    int n = 4; // number of cities (delivery points)
    vector> dist = {
        {0, 20, 42, 35},
        {20, 0, 30, 34},
        {42, 30, 0, 12},
        {35, 34, 12, 0}
    };
    vector> dp(1 << n, vector(n, -1));
    cout << "Minimum route cost: " << tsp(1, 0, dist, dp, n) << endl;
    return 0;
}

Flipkart Business Case

Customer Reviews Search

üìå Business Case

Flipkart wants to enable keyword-based review search for products so users can filter through the most relevant experiences.

üß† Algorithms Used

  • TF-IDF + Inverted Index: For indexing and retrieving relevant reviews efficiently.
  • BM25 Ranking: Better ranking for review search results.
  • Stemming / Stopword Removal: To match intent more than exact words.

üß† Intuition

Helps users quickly find what others say about specific aspects (e.g., "battery", "camera") rather than scanning hundreds of reviews.

üí• Impact

Boosts transparency, enhances user decision-making, reduces return rates by setting clear expectations.

üíª Sample Base Code (TF-IDF + Search - Python)

from sklearn.feature_extraction.text import TfidfVectorizer

reviews = [
  "Battery life is amazing on this phone",
  "Camera quality is poor",
  "Battery drains quickly",
  "Excellent performance and good battery backup"
]

query = "battery"

vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(reviews)
query_vec = vectorizer.transform([query])

from sklearn.metrics.pairwise import cosine_similarity
similarities = cosine_similarity(query_vec, X).flatten()

for i in similarities.argsort()[::-1]:
  print(f"Score: {round(similarities[i], 2)} | Review: {reviews[i]}")
